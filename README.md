# YouPiP

- 为 YouTube App启用原生画中画功能。

- 默认播放视频时上划即可开启画中画，或在影片标签栏添加按钮: 也可已在播放视频界面添加PiP按钮来激活PiP。

- 设置位于：点击头像 ~ 设置 ~ YouPiP

## 自己编译

- 使用 iOS 15 SDK 和最新的 THEOS。
